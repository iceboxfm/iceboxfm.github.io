Contact Information

Do you need to get in touch with someone regarding a feature or bug request? Do you just want to discuss the app? Here are some ways you can get in touch with the developer or other users.

Request a Feature. Is there some feature you feel is missing or would otherwise make Icebox a better app? You can submit a request for it to be considered (requires a free GitHub account).

Submit an Issue. Did you find something wrong with Icebox? From a misaligned bit of text to a full out crash of the app, feel free to report a bug if you find one and it will be reviewed for fix in a future release (requires a free GitHub account).

Join a Discussion. View or take part in any project discussions or read the announcements (read for free, but a free GitHub account is required to post).

Disclaimer

This is a one-person operation - I will try to read every request and triage every issue as they come in, but I have a job and a life beyond working on this app so please be patient and provide as much information as possible when submitting a request.

Need Some Help?

Do you need help or want to submit a feature request? Check out the Contact page for more information on how to get help or get in touch with the developer. The Guide is also a good place for learning how to use the app.
